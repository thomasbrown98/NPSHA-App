# NPSHA-App
Streamlit application for NPSHA visualization in a Frustrum for viscous fluids.

Cavitation occurs when a fluid does not have enough Net Positive Suction Head Available at the inlet side of a pump (higher than the Net Positive Suction Head Required). This application calculates NPSHA for a viscous fluid in laminar flow through a Frustrum.
